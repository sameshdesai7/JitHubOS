{\footnotesize\ttfamily char$\ast$ itoa (\begin{DoxyParamCaption}\item[{int}]{i,  }\item[{char $\ast$}]{buffer }\end{DoxyParamCaption})}



Converts a valid integer to a string and stores it in buffer. 

\begin{DoxyAuthor}{Author}
Samesh Desai 

Noah Marner 

Jackson Monk 

Blake Wagner
\end{DoxyAuthor}
First, this function checks if the number is negative and sets the negative flag according to the result. Next, if the number is 0, we set the buffer to 0, increment position, add a null terminator, and exit. While i != 0, the remainder of i \% 10 is found. The buffer at the current position is then set to the remainder plus the \textquotesingle{}0\textquotesingle{} character to calculate the number character, then i is divided by 10 to go to the digit. After the loop, the number is in reverse order; we add a \textquotesingle{}-\/\textquotesingle{} character to the end if the number is negative. Last, reverse the string and add a null terminator.
