void alarmExecution ( )

The process that executes the alarm. This will trigger when the "time" parameter is greater than or equal to the current time.

Author
Jackson Monk
Noah Marner
Sam Desai
Blake Wagner

When function is called, gets and formats system time and gets the alarm time from the currently operating processes alarm struct. If the alarm time is <= the system time, the alarm will trigger and display the message. Otherwise, calls sys_req(idle) to idle the process.
